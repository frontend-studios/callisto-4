# Callisto 4

This is the official **Callisto 4** plugin developed by plentymarkets. **Callisto 4** is the default template for [plentymarkets 7](https://www.plentymarkets.eu/tour/) online stores.

## Requirements

This is a plugin for [plentymarkets 7](https://www.plentymarkets.com). The plentymarkets [LayoutCore](https://github.com/plentymarkets/layout-core-4) plugin is required for running the plugin.

## Installing

For detailed information about plugin provisioning refer to [plentymarkets developers](https://developers.plentymarkets.com/dev-doc/basics#plugin-provisioning).
